# FolioFrame AgencyFrame Playbook

Status: agency delivery playbook. It does not create client workspaces or activate integrations.

## Agency Use Case

AgencyFrame helps agencies package FolioFrame as a repeatable buyer-access reliability service for digital product clients.

Best fit:

- agencies serving course creators
- agencies serving digital product sellers
- teams managing launches for multiple clients
- implementation partners who need handover standards
- support teams that need repeatable access recovery SOPs

## Client Workspace Model

Each client should have a separate workspace with:

- client scope summary
- product/tier matrix
- EntitlementFrame rules
- DeliveryProof test plan
- InboxGuard copy inventory
- AccessHub route
- AccessDesk issue log
- ProofMetrics baseline
- owner action register
- handover packet

Workspace rules:

- do not mix client records
- use client-approved access roles
- keep production credentials outside repository docs
- record every owner approval
- archive completed handover packets according to client policy

## White-Label Delivery Process

1. Intake and scope approval.
2. Product/tier inventory.
3. Access lifecycle mapping.
4. Delivery asset review.
5. Support route design.
6. DeliveryProof testing.
7. ProofMetrics baseline.
8. Handover packet.
9. Monthly care recommendation.

White-label note:

- FolioFrame can be delivered under an agency service wrapper when contractually approved.
- The agency should still preserve the operating standards, QA evidence and owner approvals.

| Product platform | Client product | Implementation lead | Yes | Product/tier access rules |

## Handover Templates

Use:

- DeliveryPack handover template
- DeliveryProof test plan
- AccessGuard monthly summary
- LaunchRoom issue log
- MigrationFrame cutover plan when relevant
- ProofMetrics monthly report

## AccessHub Per Client

Each client should have:

- support intake route
- issue categories
- lost access route
- wrong-email route
- broken link route
- product mismatch route
- owner approval rule
- internal escalation path

## ProofMetrics Per Client

Track:

- delivery proof completeness
- access exception count
- support request themes
- owner action status
- broken link incidents
- migration or launch recheck triggers
- monthly recommendations

## Monthly Agency Care Plan

Monthly care can include:

- link checks
- support request review
- entitlement exception review
- owner action register update
- ProofMetrics summary
- handover updates
- next-month recommendations

## Pricing Notes

AgencyFrame can support larger implementation pricing when:

- multiple client workspaces are needed
- client-specific handover packs are required
- launch support is included
- migration mapping is included
- monthly care is included
- agency team training is included

Pricing should stay tied to approved scope, client count, implementation complexity and handover depth.
