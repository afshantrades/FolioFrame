# FolioFrame AccessGuard Monthly Retainer SOP

Status: monthly service workflow for approved retainer clients. It does not add live monitoring or live integrations to the static MVP.

## Purpose

The AccessGuard monthly retainer keeps buyer access routes, support recovery paths, entitlement exceptions and operational reporting under regular review.

## Monthly Workflow

### 1. Link Checks

Review:

- checkout links
- access instruction links
- product platform links
- file delivery links
- video access links
- AccessHub support links
- onboarding links

Output:

- working links list
- broken or changed links list
- owner action list

### 2. Automation Route Review

Review approved production routes:

- checkout to entitlement
- entitlement to email
- support request to AccessDesk
- proof packet update
- reporting update

Output:

- route status
- failed or paused actions
- missing field notes
- owner review items

### 3. AccessHub Request Review

Review monthly requests by category:

- lost access
- wrong email
- broken link
- product mismatch
- delivery proof question
- refund or cancellation access question

Output:

- category counts
- unresolved requests
- repeated support themes
- owner-approved recovery route changes

### 4. Support Issue Themes

Identify:

- repeated buyer questions
- unclear access instructions
- confusing product tiers
- delivery assets that need clearer labels
- support snippets that need owner approval

Output:

- support theme summary
- recommended SOP updates
- owner copy review list

### 5. EntitlementFrame Exceptions

Review:

- bundle mismatches
- upgrade paths
- downgrade paths
- refund-sensitive cases
- cancellation-sensitive cases
- wrong-email decisions
- manual access overrides

Output:

- exception register
- approved rule changes
- pending owner decisions

### 6. ProofMetrics Report

Include:

- delivery proof completeness
- access exception count
- support request themes
- open owner actions
- closed owner actions
- recheck triggers
- recommendations for next month

Output:

- monthly ProofMetrics summary
- owner action register
- handover note if ownership changed

### 7. Owner Action Review

Review:

- open actions from previous month
- newly created actions
- overdue decisions
- production changes needing approval
- owner notes

Output:

- updated owner action list
- due state
- priority
- next review date

## Recheck Triggers

Run an off-cycle recheck when:

- a product launches
- a checkout page changes
- a price id or variant id changes
- a file folder is moved
- video access changes
- support volume rises
- email sequence copy changes
- refund or cancellation policy changes
- migration cutover happens
