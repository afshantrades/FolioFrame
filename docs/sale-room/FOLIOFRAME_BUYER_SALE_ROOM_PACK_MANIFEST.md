## Excluded From This Pack

The export pack intentionally excludes:

- Source application code.
- `node_modules/`.
- `.next/`.
- `.git/`.
- `.env` and `.env.local`.
- Production credentials.
- Database volumes.
- Rejected generated assets.

## Static/Demo Disclosure

FolioFrame is packaged here as a static service website, static portal/demo surface, implementation methodology and future service-led SaaS foundation.

It should not be represented as an operating live SaaS, live billing system, customer database, production support desk, live monitoring surface or automated fulfilment platform.

## Buyer Diligence Notes

Before acquisition close or production use, buyer should complete:

- IP and approved asset rights review.
- Trademark and domain diligence.
- Third-party dependency license review.
- Legal/privacy review.
- Production auth, backend, integration and security review if live SaaS operation is desired.
