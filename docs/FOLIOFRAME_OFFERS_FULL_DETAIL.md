# FolioFrame Offers Full Detail

This document summarizes the expanded offer model added in Step 2.

The source of truth is `content/folioframeOffers.ts`.

## Offer Schema

Every offer now includes:

- `slug`
- `name`
- `priceRange`
- `bestFor`
- `outcome`
- `includedModules`
- `deliverables`
- `timeline`
- `notIncluded`
- `ownerResponsibilities`
- `upsellPath`
- `safeScopeBoundary`
- `ctaLabel`
