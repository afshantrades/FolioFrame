# FolioFrame Advanced Modules

## EntitlementFrame

Documents entitlement logic for bundles, tiers, cohorts and exceptions.

Public route: `/entitlementframe`
Portal route: `/app/entitlementframe`

## InboxGuard

Maps delivery messages, fallback routes and support resend paths.

Public route: `/inboxguard`
Portal route: `/app/inboxguard`

## AccessHub

Provides the central static portal model for access documentation and handover.

Public route: `/accesshub`
Portal route: `/app/accesshub`

## LaunchRoom

Organizes launch readiness, rehearsal scenarios, triage notes and owner approvals.

Public route: `/launchroom`
Portal route: `/app/launchroom`

## MigrationFrame

Plans product, buyer access, delivery and support continuity during system moves.

Public route: `/migrationframe`
Portal route: `/app/migrationframe`

## ProofMetrics

Defines operational reporting for delivery proof, access exceptions and support trends.

Public route: `/proofmetrics`
Portal route: `/app/proofmetrics`

## AgencyFrame

Packages FolioFrame methods into a multi-client agency operating model.

Public route: `/agencyframe`
Portal route: `/app/agencyframe`

## FolioFrame Verified Delivery Standard

Defines the mapped, tested, documented and owner-approved standard for checkout-to-customer systems.

Public route: `/verified-delivery-standard`
Portal route: `/app/verified-delivery`
