# FolioFrame Red Flag Disclosure

## Required Buyer Disclosure

FolioFrame is currently a buyer-ready static service website and static portal/demo surface.

It is not currently an operating live SaaS, live billing platform, payment processor, customer database, support desk, monitoring system or automated fulfilment engine.

## Not Live

The following are not live production capabilities:

- Live billing
- Live Stripe checkout
- Live webhooks
- Production auth
- Real customer data
- Live support tickets
- Live email sending
- Live monitoring
- Automated fulfilment

## Demo Data

Portal content is static or fictional demo data.

Do not represent portal rows, dashboards, metrics or owner-action queues as real customer records.

## Legal And Privacy

Before commercial launch, buyer should complete:

- Legal review.
- Privacy review.
- Terms review.
- Refund-policy review.
- Data processing review.
- Customer support and deletion process review.

## IP And Brand

Before closing, buyer should confirm:

- Logo and visual asset rights.
- FolioFrame trademark availability.
- Domain ownership.
- Third-party license obligations.
- Any contributor assignment requirements.

## Safe Statement

FolioFrame is ready to review, present and deploy as a static service website and static portal/demo.

It is not ready to operate as live software until production systems and diligence are completed.
